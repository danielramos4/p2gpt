US01: Registre Client

Com a client no enregistrat

Vull registrar-me a l'aplicació amb les dades corresponents

Per a ser client de imUB i accedir als continguts de l’aplicació

Acceptance Criteria US 1.1: Registre Client: Identificador de client (correu) ja existent

EN EL CAS QUE En cas que el correu d'usuari ja estigui registrat

QUAN l'usuari seleccioni per registrar-se

EL SISTEMA mostra missatge d'error "Client ja existent en el Sistema".

El sistema dona opció d'enregistrar però el seu correu ja existeix en la llista de clients

Example del test d'acceptació US1.1
Posem un client ja existent: ajaleo@gmail.com el sistema hauria de donar Client ja existent en el Sistema.

Acceptance Criteria US 1.2: Registre Client: Contrassenya no segura

EN EL CAS QUE la contrasenya no sigui segura

QUAN l'usuari seleccioni per registrar-se

EL SISTEMA mostra missatge d'error "Contrasenya no prou segura".

El sistema dona opció d'enregistrar però la contrassenya no és prou segura

Example del test d'acceptació US1.2

Posem un usuari qualsevol no existent ajaleonou@gmail.com i una contrassenya no segura: werty, el sistema hauria de donar Contrassenya no prou segura.

Posem un usuari qualsevol no existent ajaleonou@gmail.com i una contrassenya segura: Qwertyuiop9, el sistema hauria de donar Contrassenya segura.

Acceptance Criteria US 1.3: Registre Client: Correu electronic no vàlid

EN EL CAS QUE el correu electrònic no sigui vàlid

QUAN l'usuari seleccioni per registrar-se

EL SISTEMA mostra missatge d'error "Contrasenya no prou segura".

El sistema dóna l'opció d'enregistrar però el nom no té el format d'un correu electrònic

Posem un usuari qualsevol amb un format incorrecte en el mail ajaleonougmail.com , el sistema hauria de donar Correu en format incorrecte.

Posem un usuari qualsevol amb un format incorrecte en el mail ajaleonou@gmailcom, el sistema hauria de donar Correu en format incorrecte.

Posem un usuari qualsevol amb format correcte ajaleonou@gmail.com, el sistema hauria de donar Correu en format correcte.

Acceptance Criteria US 1.4: Registre Client: Registre correcte

EN EL CAS QUE no hi hagi cap dada invàlida

QUAN l'usuari seleccioni per registrar-se

EL SISTEMA mostra el missatge “Registre vàlid”.

El sistema dona opció d'enregistrar i els registres són correctes
